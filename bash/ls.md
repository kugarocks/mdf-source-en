# Intro

List directory contents.

```bash
ls
```

---

## Long Format

With Human Readable Size.

```bash
ls -lh
```

---

## Only Directories

```bash
ls -ld */
```

Include hidden directories.

```bash
ls -ld .*/ */
```

---

## Specific Directory

```bash
ls -ld /usr
```

---

## Show Inode Number

```bash
ls -i foo.txt
```
