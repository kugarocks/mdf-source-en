# Intro

Change file owner and group.

---

## Change Owner

```bash {copyable}
chown user file
```

---

## Recursively Change Owner

```bash {copyable}
chown -R user dir
```

---

## Change Owner & Group

```bash {copyable}
chown user:group file
```

---

## Recursively Change Owner & Group

```bash {copyable}
chown -R user:group dir
```

---

## Change Group Only

```bash {copyable}
chown :group file
```
