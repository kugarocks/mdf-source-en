# Intro

Change file owner and group.

---

## Change Owner

```bash
chown user file
```

---

## Recursively Change Owner

```bash
chown -R user dir
```

---

## Change Owner & Group

```bash
chown user:group file
```

---

## Recursively Change Owner & Group

```bash
chown -R user:group dir
```

---

## Change Group Only

```bash
chown :group file
```
