# Intro

Report file system disk space usage.

```bash
df
```

---

## Readable Format

```bash
df -h
```

---

## Free Inodes Statistic

```bash
df -ih
```

---

## Specify File System

`ext4` is the file system type of Linux.

```bash
df -h -t ext4
```

`apfs` is the file system type of macOS.

```bash
df -h -t apfs
```
